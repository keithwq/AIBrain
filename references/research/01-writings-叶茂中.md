# 01 - 著作与系统思考

## 生平概览

叶茂中（1968-2022），江苏泰州人，中国著名营销策划专家，叶茂中营销策划机构创始人。以"冲突理论"闻名，服务过超200家企业，创造众多知名案例。其标志性形象是棒球帽+黑框眼镜。

## 出版著作

### 1. 《广告人手记》（1996）
- 叶茂中的第一本著作，大陆出版的第一本由中国人自己编写的偏重实务操作的广告书籍
- 出版后第一个月就卖了一万多册，叶茂中因此在广告界小有名气
- 记录他从业初期的广告实战经验与行业观察
- 提出"广告不是艺术，做广告是卖货"的核心立场
- 影响了一代中国广告人

**章节框架（据多方引用整理）：**
- 广告人的自我修养——心态、学习、积累
- 如何接单——从客户需求到创意提案
- 广告创意的实操方法——标题、文案、视觉
- 媒介策略——预算分配与投放技巧
- 客户关系管理——如何管理期望值
- 广告效果评估——什么算"有效"
- 中国广告业的现状与未来预测
- 广告人的生存法则——在理想与现实之间

### 2. 《转身看策划》（1998）
- 从广告执行向营销策划的战略转型宣言
- 提出策划的"全局观"——不能只盯广告，要看整个营销系统
- 强调市场调研对策划的基础性作用
- 提出策划人的"转身"思维：从"做什么广告"到"做什么品牌"
- 核心论点：广告是营销的一部分，营销是商业的一部分，商业是战略的一部分

**关键框架（据资料整理）：**
- 策划的"五看"：看行业、看对手、看自己、看消费者、看趋势
- 从"术"到"道"的升级路径
- 策划执行的PDCA循环模型

### 3. 《营销的12个方法》（2003）
- 系统总结其营销方法论
- 12个方法涵盖品牌定位、渠道策略、传播战术等
- 强调"市场是打出来的，不是等出来的"
- 提出"好名字是成功的一半"——命名即战略

**12个方法（据行业引用整理）：**
1. **品牌命名法**——好名字值一个亿，命名是品牌战略第一步
2. **定位法**——找到品类中的独特位置，要么第一要么唯一
3. **冲突挖掘法**——发现消费者未被满足的冲突
4. **集中投放法**——打透一个点比铺五个面更有效
5. **符号浓缩法**——将品牌浓缩为超级符号
6. **口号切割法**——一句话说清楚品牌价值
7. **品类创新法**——开辟新品类，做品类第一
8. **借势营销法**——借热点、借名人、借文化势能
9. **渠道爆破法**——打通关键渠道，实现指数级增长
10. **价格锚定法**——用价格锚点建立价值感知
11. **情感绑定法**——从功能到情感的升级
12. **持续迭代法**——营销不是一次性工程

### 4. 《冲突》（2017）
- 集大成的代表作，完整阐述"冲突理论"
- 核心论点：**营销的本质是解决冲突**，而非满足需求
- 需求是显性的、可表达的；冲突是隐性的、需要被洞察的
- 官方网站定义："冲突是战略第一步：营销的本质是洞察需求，需求隐藏在消费者冲突里。发现冲突，就是发现需求，营销就能一招致命，实现指数级增长；制造冲突，就是创造需求，营销就能改变赛道，重构市场。"

**冲突理论完整框架：**

**A. 三个冲突层级：**
- **表层冲突**——产品功能层面的冲突（如手机要续航长还是要轻薄）
- **中层冲突**——社会角色层面的冲突（如职场人想陪家人又必须加班）
- **深层冲突**——人性层面的冲突（如人想健康又贪图口腹之欲）

**B. "左右脑冲突"理论模型：**
- **左脑**：理性、价格、功能、性价比、数据、参数
- **右脑**：感性、审美、面子、情感、身份、虚荣
- **营销要做的是：用右脑的理由，让左脑买单**
- 消费者永远在左右脑之间博弈：左脑说"太贵了"，右脑说"但值得"

**C. 三源冲突：**
- **信息冲突**：知道和不知道之间的冲突（信息不对称）
- **利益冲突**：想要和能要之间的冲突（欲望与能力的落差）
- **认知冲突**：认为和/对认为之间的冲突（认知差异）

**D. 冲突的三种形态：**
- **自我冲突**：个人内心的矛盾（想吃怕胖、想买怕贵、想休息怕落后）
- **人际冲突**：人与人的矛盾（送礼物选什么、聚会上喝什么）
- **群体冲突**：群体间的矛盾（地域、代际、圈层、阶级）

**章节体系（据多方引用推测）：**
- 第1章：为什么是冲突而不是需求
- 第2章：冲突的心理学基础
- 第3章：发现冲突的方法论
- 第4章：创造冲突的策略
- 第5章：冲突视角下的品牌定位
- 第6章：冲突与广告创意
- 第7章：冲突与媒介策略
- 第8章：冲突与品类创新
- 第9章：冲突理论的实战案例
- 第10章：未来营销的冲突视角

**核心公式：冲突→焦虑→动机→购买**

## 核心理论体系

### 冲突理论（冲突营销理论）
叶茂中的理论基石，概括为：

> **"营销的本质不是满足需求，而是发现并解决冲突。"**

**理论框架：**

1. **为什么是冲突而不是需求？**
   - 需求人人都有，但解决需求的方案已饱和
   - 消费者嘴上说的需求不一定是真实动机
   - 冲突产生焦虑→焦虑产生动机→动机驱动购买
   - 好的营销品牌 = 帮消费者解决内心冲突的方案

2. **左右脑博弈模型**
   - 左脑：价格敏感、功能比较、理性计算
   - 右脑：情感满足、身份认同、审美愉悦
   - 营销方案必须同时击中左右脑

3. **冲突视角下的品牌定位**
   - 大品牌解决大冲突，小品牌解决小冲突
   - 没有冲突就没有品牌存在的意义
   - 品牌 = 冲突解决方案的载体
   - 所有伟大品牌，都是伟大冲突的伟大解决方案

### 品牌三度理论
- **知名度**：让消费者知道你（广告解决）
- **美誉度**：让消费者说你好（产品/服务解决）
- **忠诚度**：让消费者非你不可（情感绑定解决）

### 市场=人口×购买力×购买欲望
叶茂中的经典公式，强调：
- 人口结构和城市化是市场机会的第一变量
- 购买力提升意味着消费升级的赛道
- 购买欲望靠营销激发

### 12字品牌方针
- **找对人**：精准目标人群
- **说对话**：用消费者的语言沟通
- **做对事**：落地执行要匹配战略

### 大赌大赢哲学
- "资源有限，必须集中。打透一个点，比铺五个面更有效。"
- "与其更好，不如不同。与其分散，不如聚焦。"
- "大创意=大投入=大回报"
- "不要总想创新，先把对的做一遍。赌对了，放大；赌错了，认。"
- "没有预算就做减法，不要做加法。"

## 关键概念与术语

| 术语 | 释义 |
|------|------|
| **冲突** | 未被满足的欲望与现实之间的落差，购买行为的底层驱动力 |
| **左右脑博弈** | 消费者在做决策时理性（性价比）与感性（面子/情感）的交战 |
| **痛点** | 消费者遭遇的具体问题（叶茂中认为痛点只是冲突的表层） |
| **痒点** | 消费者向往但未满足的欲望 |
| **冲突营销** | 以发现和解决消费者冲突为核心的营销方法论 |
| **大创意** | 能一句话说清楚、让受众瞬间记住的核心传播概念 |
| **符号化** | 将品牌浓缩为一个视觉/听觉符号，降低记忆成本 |
| **集中投放** | "打透一个点比铺五个面更有效"的媒介策略 |
| **降维打击** | 用更高维度的冲突方案碾压竞争对手的思维 |
| **卖货哲学** | 广告的唯一目的是卖出产品，而非获奖或讨好观众 |
| **洗脑式重复** | 高频重复品牌信息以强化记忆，争议极大但有效 |

## 真信念（贯穿3+语境的核心信念）

1. **"广告不是艺术，广告是卖货"**
   - 出现在《广告人手记》《冲突》以及多次演讲中
   - 反对华而不实的创意，推崇结果导向的营销

2. **"没有冲突就没有营销"**
   - 从早期方法论到《冲突》专著，再到晚年演讲，始终坚持
   - 是其整个理论体系的元命题

3. **"大创意就是'一句话说不清楚就不是好创意'"**
   - 出现在多本书籍、访谈、培训中
   - 强调传播的极简原则

4. **"市场是打出来的，不是等出来的"**
   - 贯穿其职业生涯，从创业到操盘大品牌案例
   - 行动主义精神，反对纸上谈兵

5. **"品牌是冲突解决方案"**
   - 核心信念，贯穿《冲突》全书及后续延展
   - 定义了叶茂中看品牌的方式

6. **"好名字值一个亿"**
   - 多次在案例复盘和演讲中强调
   - 认为命名是品牌战略的第一步
