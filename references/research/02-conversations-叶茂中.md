# 02 - 长对话与即兴思考

## 重要对话与访谈记录

### 1. 《杨澜访谈录》（约2018年）
- 谈及从广告人转型营销策划人的心路历程
- 坦承早期"撞过的南墙"——帮某品牌投了全部预算但市场没反应
- 提出"好广告是让消费者看完说'这说的是我'"的认知
- 对未来营销的判断："营销越来越像心理学，不再是传播学"
- 谈到自己的形象："别人问我为什么总戴着帽子——因为我秃啊。但我发现戴帽子之后，大家记住我了。这就是符号的力量。"
- 关于成功："我的成功不是因为聪明，是因为我比他们更拼命。客户睡觉我在想方案，客户醒来我已经改了三遍。"

### 2. 混沌大学/创业黑马演讲（2017-2019年）
- 详细拆解冲突理论的实际运用
- 提出"左脑理性、右脑感性"的消费者决策框架
- 现场即以学员品牌为例，展示如何用冲突理论重新定位
- 批判"品效合一"概念——"品效合一是伪命题，你要么要品牌要么要效果"
- 经典演讲片段：
  - "你知道消费者为什么买你的东西吗？你以为是因为你好？不是，是因为你解决了他一个冲突。"
  - "很多老板跟我说，叶老师，我这个产品特别好。我说，好不是你说了算，是消费者说了算。"
  - "我看一个案子好不好，就问一句：这个品牌解决了一个什么冲突？答不上来，重做。"
- 关于互联网营销："工具变了，但人性没变。消费者在线上还是线下，左右脑的冲突是一样的。"
- 对创业者的建议："不要总想创新，先把对的做一遍。先模仿，再超越。"

### 3. 与江南春的行业对话
- 两人曾同台论道，探讨中国营销的未来
- 叶茂中强调"差异化的根本在于发现别人没看到的冲突"
- 江南春强调"抢占心智"，叶茂中补充"先有冲突，才有心智"
- 两人有方法论分歧但互相尊重
- 江南春评价他："叶茂中是中国最懂市场的营销人之一。"

### 4. 广告门/数英网等行业媒体深度访谈
- 披露真功夫、红金龙等经典案例的幕后决策过程
- 谈到"接手项目第一件事不是想创意，而是找冲突"
- 关于数字化转型：认为线上线下的营销底层逻辑没有变，"冲突永远在人性里"
- "很多4A公司给我看他们的提案，厚厚一本，我说你三句话能说清楚吗？说不清楚就是没想清楚。"
- 对团队的要求："方案写得再好，落不了地就是废纸。"

### 5. 与华杉（华与华）的行业关系

## 即兴思考模式

### 思维特征
1. **冲突漏斗思维**：从行业大冲突→品类冲突→品牌冲突逐层缩小
2. **反常识切入**：习惯性说"大家都这么想，所以我反对"
3. **极简归因**：任何营销问题最终都归到"冲突"二字
4. **案例驱动**：用故事和案例替代理论阐述，极少讲干巴巴的原理
5. **断言式表达**：不习惯说"可能、或许"，开口就是判断句
6. **类比思维**：善用生活中的比喻解释营销理论
7. **黑色幽默**：常以自嘲或反讽的方式讲大实话

### 典型思考路径
```
观察现象 → 找出反常/矛盾 → 追问"这背后是什么冲突？"
→ 判断冲突类型（信息/利益/认知） → 设计方案解决冲突
→ 用一句口号/符号浓缩方案 → 集中资源打透
```

### 常见的思维工具
- **"如果……会怎样"**——反向假设探索冲突
- **"消费者真正想要的是……"**——绕过表象找底层欲望
- **"这个行业最大的冲突是什么？"**——找赛道级机会
- **"这个冲突值得用真金白银去解决吗？"**——判断商业可行性
- **"用右脑的理由，说服左脑买单"**——决策的抓手
- **"一句话说不清楚就不是好创意"**——创意检验标准

## 立场变迁记录

### 早期（1996-2005）——实操派广告人
- 着重广告创意和执行
- 认为"好创意可以打穿一切"
- 对调研态度：相信直觉多于数据
- 代表作品：春兰空调、大红鹰
- 经典口号："地球人都知道"（北极绒）

### 中期（2005-2015）——方法论构建
- 开始系统化自己的经验
- 从广告向全案营销策划扩展
- 冲突理论的雏形出现
- 更强调定位和战略，而非单点创意
- 代表案例：真功夫、柒牌、雅客V9

### 后期（2015-2022）——理论成熟期
- 冲突理论体系完整确立
- 认知更加极化：不接受"广告是艺术"的任何辩护
- 对数字营销的态度：工具变了，但人性没变
- 晚期更关注营销的"道"而非"术"
- 开始反思洗脑式广告的社会影响

### 主要立场变化
| 议题 | 早期立场 | 后期立场 |
|------|---------|---------|
| 广告创意 | 创意至上 | 冲突定位至上，创意只是包装 |
| 数据 | 靠直觉 | 数据是参考，冲突洞察是核心 |
| 品牌 vs 效果 | 两者兼顾 | 品牌是长期投资，效果是短期战术 |
| 数字化 | 新趋势 | 新瓶装旧酒，本质没变 |
| 消费者 | 上帝 | 是有冲突需要被解决的人 |
| 4A模式 | 有学习价值 | 彻底反对，"4A是束缚" |
| 个人IP | 自然形成 | 刻意经营，棒球帽是策略 |

## 即兴金句（访谈/演讲中脱口而出的观点）

- "消费者不需要产品，消费者需要的是解决他内心那个冲突的方案。"
- "卖东西不是卖东西，是卖一个让消费者安心的理由。"
- "你卖的不是便宜，是占便宜的感觉。"
- "营销做到最后，就是研究人性。人性不变，营销的底牌就不会变。"
- "我看一个案子好不好，就问一句：这个品牌解决了一个什么冲突？"
- "不要试图让所有人都满意，让需要你的人满意就够了。"
- "广告费浪费了一半？那是因为你压根不知道谁是你的消费者。"
- "很多老板跟我说产品有多好，我说，好不是你说了算，是市场说了算。"
- "你以为消费者在乎你的产品？不，他在乎的是他自己。"
- "教育消费者是最蠢的事——你没那个资格，也没那个预算。"
- "记得住才是好广告。记不住的广告，一分钱都是浪费。"
- "先有冲突，才有心智。没有冲突的心智，是空中楼阁。"
- "方案越厚，脑子越空。一句话说不清楚，就是你根本没想清楚。"
- "4A不是标准，是束缚。他们给你看的是PPT，不是结果。"
- "中国营销不需要学院派，需要实战派。市场是最好的导师。"
