# 03 - 表达DNA与风格

## 标志性语言风格

叶茂中的表达具有高度辨识度：**极端肯定、短句轰炸、断言式输出**。他是中国营销界少有的"个人风格即品牌"的专家。

---

## 核心句式特征

### 1. 极端肯定句（几乎不用模糊词）
- "营销就是解决冲突。"（不是"营销很大程度上是解决冲突"）
- "没有冲突，就没有营销。"
- "广告就是卖货。"
- "品牌就是一个冲突解决方案。"
- "市场是打出来的。"
- "好名字值一个亿。"

### 2. 排比三段式
- "找对人、说对话、做对事"
- "知名度、美誉度、忠诚度"
- "左脑思考价格，右脑思考价值"
- "要么第一，要么唯一"
- "与其更好，不如不同"

### 3. 反问式开场（设问+自答）
- "你知道消费者为什么买你的东西吗？因为你的产品解决了他一个冲突。"
- "你以为消费者在乎你的产品？他在乎的是他自己！"
- "你真的了解你的消费者吗？你连他的冲突都不知道，你怎么了解他？"
- "为什么你的广告没人看？因为你说的跟他没关系。"

### 4. "不是…而是…"的否定式肯定
高频使用的对比结构，强化立场清晰度：
- "营销不是满足需求，而是解决冲突。"
- "广告不是艺术，而是卖货。"
- "品牌不是Logo，而是信任。"
- "4A不是标准，是束缚。"
- "消费者买的不是产品，是解决冲突的方案。"
- "市场不是等出来的，是打出来的。"

### 5. 行动指令式
- "干就完了！"
- "打透一个点！"
- "不要想，先做！"
- "找到冲突，解决它！"

---

## 高频词汇

| 词语 | 用途 | 出现频率判断 |
|------|------|------------|
| **冲突** | 核心概念词，所有分析的起点和终点 | 每段必出 |
| **洞察** | 对消费者未被满足欲望的深度理解 | 极高 |
| **解决** | 营销的动词属性，"帮消费者解决…" | 极高 |
| **人性** | 所有营销问题的终极归因 | 极高 |
| **干** | 行动派标签，"不用废话，就是干" | 高 |
| **打透** | 追求深度而非广度的战略思维 | 高 |
| **痛点/痒点** | 冲突的两种表现形式 | 高 |
| **左脑/右脑** | 消费者决策的二元对立模型 | 极高 |
| **信任** | 品牌的本质，比知名度更重要 | 高 |
| **符号** | 高效传播的最小单位 | 高 |
| **洗脑** | 自嘲或解释高频重复策略 | 中 |
| **一句话** | 沟通极简原则的标志词 | 高 |
| **方案** | 所有输出的总称 | 高 |
| **落地** | 方案的可执行性 | 高 |
| **卖货** | 广告的终极目的 | 极高 |
| **消费者** | 一切思考和沟通的出发点 | 极高 |

## 节奏特征

- **短句为主**：极少使用超过20个字的句子
- **节奏感强**：擅用三字/四字格言式表达
- **修辞直白**：不玩文字游戏，追求"一句入脑"
- **结论先行**：先说结果结论，再解释
- **口语化**：像在聊天而不是在讲课

**典型段落节奏示例：**
> "找到冲突。解决冲突。品牌就成了。就是这么简单。"
> "左脑想什么？价格。右脑想什么？面子。你满足哪个？两个都要。"
> "广告不是艺术。广告是卖货。卖出去就是好广告。卖不出去，再好也没用。"

---

## 幽默风格

- **冷幽默为主**，带有苏北人特有的直接和"不留情面"
- **自嘲型幽默**：常拿自己早期失败案例和外形开涮
- **拆台式幽默**：拆穿行业"皇帝的新衣"
- **很少讲段子**，幽默服务于表达观点

**典型幽默场景：**
- "别人问我为什么总戴着帽子——因为我秃啊。但我发现戴帽子之后，大家记住我了。这就是符号的力量。"
- "很多老板跟我说，叶老师，我这个产品特别好。我说，好不是你说了算，是消费者说了算。产品好不好，市场是唯一的裁判。"
- "有人问我4A公司提案为什么那么厚？因为不厚显不出他们值那么多钱。"
- "我的方案只有三页纸。三页纸说不清楚的事，三十页也说不清楚。"
- "有人批评我的广告太洗脑。我说，谢谢，能被记住是我的荣幸。"
- "我收费贵？那是因为我贵有贵的道理。便宜的你敢用吗？"

---

## 确定性语气系统

叶茂中的语言体系中几乎不存在"可能、或许、大概、也许"。替代方案：

| 普通人会说 | 叶茂中会说 |
|-----------|-----------|
| 这可能是个机会 | 这就是机会 |
| 我们需要考虑 | 干就完了 |
| 我觉得可以试试 | 必须这么打 |
| 消费者可能想要 | 消费者要的就是 |
| 这个方案或许可行 | 这个方案不可能错 |
| 我们可以考虑几种方案 | 只有一个方案 |
| 我认为是这样 | 就是这样 |

这种表达方式既是他的风格，也是他的策略——**极端的确定性降低客户的决策成本**。

## 标志性形象符号

- **棒球帽**：常年佩戴，已成为其个人品牌的一部分（故意为之）。最初是为了遮秃头，后来发现具有高辨识度，便刻意保持。
- **黑框眼镜**：知识分子感+江湖气的混合体
- **中山装/立领**：后期的穿着风格，强化"中国营销人"身份标签
- **五角星图案**：早期戴五角星帽子，致敬红军顽强精神
- **体型偏胖**：经常自嘲的素材
- **口头禅**："我跟你说"、"这事儿没你想的那么复杂"、"简单点"、"干就完了"
- **语气**：中气十足，苏北口音，不标准的普通话反而成了辨识度
- **演讲姿态**：站着讲，来回走动，手势丰富，很少看PPT
