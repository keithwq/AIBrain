交付向导

草稿不会扣积分

交付向导

把复杂任务拆成几步，降低老师填写负担，也让结果稳定。

1 确定交付物教案、课件、讲义、逐字稿
2 填写课堂条件年级、时长、课型、学生问题
3 放入材料作文题、样文、教材页、校区要求
4 生成并微调右侧预览结构，底部继续追问

生成一节作文升格课

材料袋
可放优秀文、问题文、上次课堂反馈、教材页或校区模板。

将生成的交付包

一、课堂主线 从“事情发生了”推进到“人发生了变化”，用一个学生能懂的核心问题贯穿。
二、四段教学流程 导入、样文诊断、现场升格、迁移练习，每段带教师话术和学生任务。
三、板书与 PPT 大纲 板书保留 3 个关键词，PPT 只做课堂推进，不堆满理论。
四、课后修改任务单 学生回去只改一件事，老师下次能检查，不让任务失控。
