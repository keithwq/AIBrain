任务树

家庭菜谱 / 今日版

生成温和清淡的一日三餐

以普通食材、合理搭配和可执行做法为主，不承诺调理疾病，不替代营养师或医生。

今日餐桌主线

少油、温热、不过甜；主食粗细搭配，晚餐减轻负担。把“养生感”藏进家常菜里。

一日食谱

早

小米南瓜粥 + 蒸山药 + 水煮蛋。口味淡，适合早上不想吃重的人。

午

香菇鸡丝面 + 清炒小白菜。保证蛋白质，汤面不过油。

晚

莲子百合银耳羹少糖版 + 清蒸鲈鱼 + 焯拌菠菜。

采购清单

小米、南瓜、山药、鸡胸肉、香菇、小白菜、百合、银耳、鲈鱼、菠菜。

材料库

人数
2 位成人 + 1 位老人
偏好
不吃辣；晚餐希望清淡；家里有小米、山药、鸡蛋、菠菜。
禁忌
不按疾病开菜单；慢病、孕期、儿童特殊饮食需走医生或营养师建议。
