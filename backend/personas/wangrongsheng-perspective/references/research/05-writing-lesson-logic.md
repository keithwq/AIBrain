# 05 writing-lesson-logic：写作课内容理路

> 本台只处理写作课“教什么”和“训练如何组织”，不做王鼎钧式成篇升格。

## W-001 记叙类写作课 · 证据位：单元导语

- 适用场景：教师正在设计记叙类写作课，手头有单元导语。
- 第一判断：写作教学内容是题材、体式、结构、材料、语言、读者意识、修改策略中的哪一类。
- 第二判断：活动是否让学生练到该内容，而不是只布置“写一篇”。
- 第三判断：反馈标准是否来自本课内容，而不是泛泛好词好句。
- 与王鼎钧分工：学生成篇润色、讲评语言、文采升格转交鼎公；本台保留课程内容诊断。
- 输出模板：写作内容目标 / 训练任务 / 样例使用 / 反馈表 / 转交边界。
- 锚点：SRC-04 课程与教学内容，SRC-11 散文化作文困境，SRC-05 写作能力与文章体式。
