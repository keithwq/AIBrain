# 王鼎钧著作体系深度分析

## 一、著作全景总览

王鼎钧创作生涯长达近90年（1939-2023），著作近40种，横跨散文、小说、剧本、文学理论、杂文、评论六大领域。本章按体系分类深度分析每一部核心著作的内容、结构和思想。

---

## 二、"人生三书"体系（1975-1976）

### 2.1 《开放的人生》（1975）

**出版信息**：尔雅出版社，1975年初版，"人生三书"第一部。

**核心定位**：励志小品文集，面向青年读者，以百余字短篇讨论做人的基本修养。偏重社会伦理与人际互动。

**完整篇目结构（70篇）**：

| 篇章 | 核心观点 | 关键金句 |
|------|---------|---------|
| 三种成长 | 成长分肉体、知识、人格三层次 | "长大不算成长" |
| 睡眠之前 | 每日睡前反省是人格成长的捷径 | "一天过后，要有一个安静的片刻" |
| 一日之计 | 早晨决定一天的方向 | "早晨的意志力最强" |
| 完人 | 完美主义是进步的敌人 | "完人不是无缺点，而是有优点" |
| 团队接力 | 个人与团队的关系如接力赛 | "不必跑完全程，但要跑好你那一棒" |
| 君子之争 | 竞争有品格高下之分 | "赢要赢得光彩，输要输得大方" |
| 合作 | 合作是现代社会的基本能力 | "没有人能独自成功" |
| 人才 | 人才的标准不在学历而在解决力 | "人才是找出来的，不是考出来的" |
| 遗珠 | 怀才不遇的真相 | "真有珍珠的光泽，不怕被埋没" |
| 邪正 | 邪正的界限 | "邪正之分，往往在第一步" |
| 苦 | 苦是人生的常态 | "不受苦，就不懂人生" |
| 六字箴言 | "不要怕，不要悔" | 全文最著名的一篇——人生两阶段六字真言 |
| 炎凉 | 世态炎凉的常态 | "人情如纸张张薄"但不必怨恨 |
| 人比人 | 攀比的危害 | "人比人，气死人" |
| 自知之明 | 认识自己的边界 | "知道自己不能做什么，比能做什么更重要" |
| 天才 | 天才与努力的关系 | "天才是长期的勤奋" |
| 人生如戏 | 扮演好自己的角色 | "戏演完了，记得下台" |
| 才命 | 才与命的辩证 | "有才还要有命，但命不否定才" |
| 信心的力量 | 信心的心理学机制 | "你认为自己能，你就真的能" |
| 与人为善 | 善意的力量 | "最大的利己是利人" |
| 恕道 | 宽恕别人也是放过自己 | "恕字是'如心'——将心比心" |
| 认识谣言 | 谣言的传播机制 | "谣言止于智者" |
| 友谊银行 | 朋友如银行存款 | "平时不存，急时取不出" |
| 代差 | 代沟是必然的 | "每一代都觉得自己是对的" |
| 因果 | 因果律不是迷信 | "种什么因，得什么果" |
| 路 | 路是人走出来的 | "地上本没有路，走的人多了也便成了路" |
| 思而后行 | 行动前要思考 | "三思而后行" |
| 慎言 | 说话的艺术 | "话到嘴边留半句" |
| 学笑记 | 假笑可以变成真笑 | "笑着笑着就真的快乐了" |
| 好话多说 | 赞美是一种能力 | "好话一句三冬暖" |
| 兴趣与钱 | 兴趣与谋生的平衡 | "兴趣最好不是赚钱的工具" |
| 创意 | 创意的本质 | "创意是旧元素的新组合" |
| 爱的迷惑 | 爱的真伪 | "爱不是占有，是成全" |
| 现代恋爱 | 恋爱观的变迁 | "恋爱自由不等于恋爱容易" |
| 求新 | 求新的限度 | "新不一定好，但好的一定新" |

**六字箴言全文**：
> 四十年前，我遇到一位老人，他送给我两句话。他说："人生的秘诀只有两句话。年轻时，六字——不要怕，不要悔。"
> 他说："年轻人，你的人生刚开始。在这几十年里，不管遇到什么困难，记住——不要怕。勇敢地去做，勇敢地去闯。等你老了，回头再看这一生，记住——不要悔。无论做了什么选择，都不要后悔。因为后悔没有用，而且——你做的每一个选择，都是当时最好的选择。"

**总评**：《开放的人生》以最平实的语言讲最深刻的道理，用"小故事讲大道理"的手法被后来无数作者效仿。"六字箴言"篇是全文最佳，流传极广。

### 2.2 《人生试金石》（1975）

**出版信息**：作者自印，1975年初版，"人生三书"第二部。

**核心定位**：比《开放的人生》更深一层，从"做人"进入"事上磨练"——强调实践中的挫折才是真正的试金石。

**核心篇章**：
- "试金石"概念：真金不怕火炼，真人不惧挫折
- "逆境是化了妆的祝福"——核心观点
- "失败不是跌倒，是跌倒后不肯爬起来"
- "小挫折过滤弱者，大挫折成就强者"
- "没有经过试炼的人生是脆弱的"

**本书特色**：从个人修养进入"社会生存"的维度，讨论如何面对不公平、背叛、失败等现实困境。

### 2.3 《我们现代人》（1976）

**出版信息**：作者自印，1976年初版，"人生三书"第三部。

**核心定位**：从"做人"、"炼人"进入"现代人的处境"——讨论在工业社会、都市化、媒体时代中如何保持人的完整性。

**核心议题**：
- "现代病"的诊断：疏离、焦虑、虚无
- "在群体中保持独立的困难"
- "信息的爆炸与智慧的贫乏"
- "速度对深度的侵蚀"
- "传统断裂后的精神漂泊"

**金句**：
- "现代人最大的痛苦不是贫穷，是不知道自己要什么"
- "我们拥有的越来越多，感觉到的越来越少"

### 2.4 "人生三书"的整体价值

三本书形成了逐步递进的结构：
1. 《开放的人生》→ 基本修养（我与我、我与人的关系）
2. 《人生试金石》→ 挫折教育（我与困难的关系）
3. 《我们现代人》→ 时代处境（我与时代的关系）

台湾总销量逾60万册，被誉为"凡有井水处，即有鼎公书"。

## 三、"作文四书"体系（1978-1986）

### 3.1 《文学种籽》（1982）

**出版信息**：明道文艺杂志社，1982年初版；2003年尔雅出版社再版。"作文四书"第二部（按写作顺序是第一部）。

**全书架构（三部分+附录）**：

**【第一部分：语言基础】**

**第一章 语言**
- 语言是人类最伟大的发明
- "文章是说话的延长"
- "话怎么说，文章就怎么写"
- 但口语和书面语有区别：口语随意，书面语严谨
- "语言是思想的直接现实"

**第二章 字**
- 字的音、形、义三要素
- "中文的每个字都是艺术品"
- 用字的精确是写作的基本功
- "一字之差，意思可以差之千里"

**第三章 句**
- 句子的结构：主语、谓语、宾语
- 长句和短句的不同效果
- "句子的节奏决定了文章的节奏"
- "短句如刀，长句如水"

**第四章 语文功能**
- 语文的三大功能：记录、描绘、判断
- 记叙文、描写文、议论文分别对应三大功能
- "一篇文章往往三种功能并用，但有一种为主"

**【第二部分：文学要素】**

**第五章 意象**
- **这是全书理论核心**：意象是文学的最小单位
- "狭义的文学还有一更狭小的核心，那就是表达心思意念要出之以'意象'"
- "文学作家所写的乃是意象。认识这个核心，才真正认识文学"
- 意象 = 意（内在情感）+ 象（外在形象）
- 意象不同于比喻——比喻是临时的，意象是结构性的
- 从《诗经》的"关关雎鸠"到现代诗的意象经营

**第六章 题材来源**
- 题材从哪里来？——从生活中来、从阅读中来、从想象中来
- "作家要有一个'题材仓库'"
- 如何建立题材仓库：观察、记忆、整理
- "最平凡的题材，在最好的作家手里可以写出最不平凡的作品"

**第七章 散文**
- 散文的定义、类别、技法
- 散文是"一切文体的基础"
- "散文不能太像散文，要像小说一样有故事，像诗一样有节奏"

**第八章 小说**
- 小说的要素：人物、情节、环境
- 小说和散文的区别
- "小说是说故事的延长"

**【第三部分：创作进阶】**

**第九章 剧本**
- 剧本是"吵架的延长"
- 冲突是戏剧的灵魂
- "剧本里没有'他说'、'她想'，只有动作和对话"

**第十章 诗**
- 诗是"唱歌的延长"
- 诗的语言：浓缩、跳跃、暗示
- "诗不是感情的宣泄，是感情的升华"

**第十一章 体裁选择**
- 如何选择适合的体裁
- "同一个题材可以用不同的体裁来写，效果完全不同"
- "体裁不是框框，是工具"

**第十二章 胎生与卵生**
- **全书最特色的概念**：
  - "胎生"：先有主题（胚胎），再找材料——适用于议论文、有明确立意的文章
  - "卵生"：先有材料（蛋），再孵出主题——适用于小说、散文等创作
- "成熟作家两种方法都会用"

**附录**：
- 猫猫虎虎
- 复活疑案

**总评**：《文学种籽》是王鼎钧创作理论的核心著作。如果只读一本他的理论书，就读这本。它以最系统的框架阐述了"文学是什么"和"如何创作文学"两个根本问题。

### 3.2 《作文七巧》（1984）

**出版信息**：作者自印，1984年初版，2003年尔雅再版。"作文四书"第三部。

**全书架构**：

**自序（极重要）**：
> "我是赤着脚走路的那种人，路上没有红毯，只有荆棘。中年以后整理自己的生活经验，发生了一个疑问：当年走在路上，前面明明有荆棘，为什么走在前面的人不告诉我呢？"
> "我恍然大悟：没有人告诉我，是因为他们也不知道。他们也许走的是另一条路，没有经过我经历的荆棘。于是，我决定把自己走过的路写下来，把哪里有荆棘、哪里有甘泉写下来，放在路旁，让后面走过来的人捡去看看。"

**第一章 记叙的技巧（直叙）**
- 直叙的定义：按照事件发生的先后顺序叙述
- "直叙是最自然的写法，也是最难写好的写法"
- 直叙的陷阱：流水账
- 如何避免流水账：有详有略、重点突出
- 案例：《桃花源记》的直叙分析
- 练习：写一件小事，只用直叙

**第二章 记叙的技巧（倒叙）**
- 倒叙的定义：从事件的中途或结尾开始写，再回头交代前因
- "倒叙是制造悬念最直接的方法"
- 倒叙的几种形式：
  1. 从结局倒溯
  2. 从中间高潮处开始
  3. 双线交错（今昔交替）
- 案例：从"我听到这个消息时"开始的倒叙写法
- 倒叙和直叙的混合使用

**第三章 抒情的技巧**
- "无情固然不能抒情，无情恐怕也不宜记叙议论"
- 抒情的本质：让读者感受到作者的情感
- 直接抒情 vs 间接抒情
- "间接抒情更有力量：不写'我很难过'，写'窗外的雨一直下个不停'"
- 借景抒情、借物抒情、借事抒情
- 案例：通过写物写景来传达情感

**第四章 描写的技巧**
- "描写，写的是景"
- '景'不限风景，包括一切"景象"
- 描写的层次：远景、中景、近景、特写
- 五感描写：视觉、听觉、嗅觉、触觉、味觉
- "描写不是拍照，是绘画——有选择、有重点、有情感"
- 案例：写一棵树（对比《那树》的描写手法）

**第五章 议论的技巧（归纳）**
- 归纳法：从具体事例推出普遍结论
- "归纳是从个别到一般"
- 归纳的案例：观察几个人→得出结论
- 归纳的陷阱：以偏概全
- "归纳要有足够的案例支撑"

**第六章 议论的技巧（演绎）**
- 演绎法：从普遍原理推出具体判断
- "演绎是从一般到个别"
- 演绎的三段论：大前提→小前提→结论
- "人都会死（大前提），苏格拉底是人（小前提），所以苏格拉底会死（结论）"
- 演绎在文章中的灵活运用

**第七章 四种写法的综合运用**
- 记叙、抒情、描写、议论在文章中的混合
- "好文章往往是四种手法交织使用"
- 一篇范文的逐句分析：哪里是记叙、哪里是描写、哪里是抒情、哪里是议论
- "综合运用不是杂烩，是有主有次的配合"

**第八章 文路无尽誓愿行（全书的跋）**
- "写作的路没有尽头"
- "七巧只是入门，真正的写作在七巧之外"
- 鼓励读者：技巧学会了，剩下的就是多读多写

**附录**：各章练习题的参考答案

**总评**：《作文七巧》可能是华语世界最实用的一本写作技法书。王鼎钧的过人之处在于：他把写作这个"说不清"的事情拆解成了七个可练习的技法模块。每一个技法都有案例、有练习、有参考。

### 3.3 《作文十九问》（1986）

**出版信息**：作者自印，1986年初版，2004年尔雅再版。"作文四书"第四部。

**全书架构**：师生问答形式，19个问题覆盖写作的深层困惑。

**十九问全景**：

**第一问：作文要起承转合吗？**
- 答：起承转合是基本框架，但不可教条
- "起要开门见山，承要顺理成章，转要出人意料，合要余味无穷"
- 重点在"转"——没有转折的文章是平的

**第二问：如何立意？**
- 立意是一篇文章的灵魂
- "立意要高，但不要高到够不着"
- 如何找到好的立意：多问"为什么"

**第三问：用什么材料？**
- 材料要服务于主题
- "宁可材料少而精，不要多而滥"
- 材料的取舍标准：是否直接支持主题

**第四问：安排材料的顺序？**
- 按时间、按空间、按逻辑
- "最好的顺序是让读者一口气读完"

**第五问：什么才是恰当的比喻？**
- "比喻是把陌生的东西变成熟悉的东西"
- 比喻要贴切、新颖、不怪异
- "陈腐的比喻不如不用"

**第六问：如何开头？**
- 开头的几种方法：以事开头、以景开头、以问开头、以警句开头
- "开头要像虎头——有气势"
- "开头是给读者的第一个承诺"

**第七问：如何结尾？**
- 结尾的几种方法：总结式、余韵式、反问式、呼应式
- "结尾要像豹尾——有力"
- "不好的结尾往往是因为你不知道在哪里停"

**第八问：文章如何分段？**
- 分段是节奏的控制
- "一段只有一个中心意思"
- "短段制造紧张感，长段制造从容感"

**第九问：句子怎样才算好？**
- 好句子的标准：准确、简洁、有节奏
- "句子要像钉子——钉在读者心里"
- 长短句交替使用

**第十问：怎样用字用词？**
- "准确的词只有一个是合适的"，法国作家福楼拜的"一词说"
- "宁用具体名词，不用抽象名词"
- "动词是句子的灵魂"

**第十一问：如何表达情感？**
- "情感要节制——太满了惹人烦，太少了不动人"
- "用事写情，不是用情写情"

**第十二问：写具体还是写抽象？**
- "写具体永远是上策"
- 如何抽象变具体：用例子、用比喻、用故事

**第十三问：简洁好还是详细好？**
- "该简则简，该详则详"
- "简不等于简陋，详不等于啰嗦"

**第十四问：怎么写得生动？**
- 生动的秘诀：具体、动作、对话
- "写得生动就是让读者看到、听到、感受到"

**第十五问：什么是好文章？**
- "好文章让人读了想哭，又哭不出来"
- "好文章的标准：准确、清晰、有力量"

**第十六问：文章的个性重要吗？**
- "个性不是刻意求来的，是写多了自然形成的"
- "模仿是必要的阶段，但不要停留在模仿"

**第十七问：为什么我写的东西没有深度？**
- "深度来自思考，不是来自词汇"
- "多思考一层，深度就多一分"

**第十八问：读诗对作文有帮助吗？**
- "读诗是培养文心最好的方法"
- "诗的语言是最精炼的语言"

**第十九问：怎样才算入门了？**
- "写够十万字废稿，才算入门"
- "入门之后还有很长的路，但至少你已经在路上了"

**总评**：《作文十九问》是"作文四书"中最亲切的一本，师生对话的形式让读者感觉有老师在身边一对一辅导。十九个问题覆盖了从技到道的几乎所有写作困惑。

### 3.4 《灵感》（1978初版/1989增订）

**出版信息**：作者自印，1978年初版；1989年尔雅出版社增订再版。

**核心内容**：一部关于"灵感"的特殊著作。王鼎钧受古人诗话笔记影响，又受现代"手记文学"启发。

**全书结构**：
- 第一部分：灵感——短篇灵感记录
- 第二部分：灵感补——更多灵感案例
- 第三部分：灵感五讲
  - 可大可久谈原型
  - 亦师亦友谈模仿
  - 成方成圆谈结构
  - 有隐有显谈比喻
  - 求新求变谈造句
- 第四部分：灵感访谈

**核心理念**：灵感不是天上掉下来的，是长期积累后偶然触及的结果。"灵感是长期积累、偶然得之"——这是全书最核心的判断。

## 四、抒情散文代表作（1970-2009）

### 4.1 《碎琉璃》（1978）

**出版信息**：九歌出版社，1978年初版。

**核心内容**：自传体散文集，以童年记忆为素材，书名"碎琉璃"象征童年记忆——美丽但易碎。

**名篇一览**：
- 《一方阳光》——追忆母亲，冬日午后阳光下的针线活
- 《红头绳儿》——童年故事
- 《那树》——城市发展中的老树命运
- 《山水》——故乡的自然记忆
- 《青纱帐》——山东乡村的高粱地
- 《我们的功课表》——战时教育的回忆

**核心段落**（《一方阳光》）：
> 一方阳光从屋檐上斜照下来，在方砖地上铺成一个斜斜的四边形。母亲坐在那方阳光里，低着头做针线。阳光照在她头发上，那头发原是黑的——不，原是花白的——在阳光里看起来像金丝一样。她的手很巧，一针一线，不紧不慢，把破碎的日子缝补起来。她一边做活一边说话，声音轻轻的，像是对我说，又像是自言自语。

### 4.2 《左心房漩涡》（1988）

**出版信息**：尔雅出版社，1988年初版。

**全书架构**：
- 大序
- 第一部：大气游虹（明灭、水心、惊生、如果、两猜、失名、山水、读江、旧曲）
- 第二部：世事恍惚（黄河在咆哮、春雨·春雷等）
- 第三部：江流石转（中国在我墙上、红石榴等）
- 第四部：万木有声（年关、园艺、夜行、脚印、言志、对联、天堂等）

**书名寓意**："中国大陆是我的左心房，台湾是我的右心房"——两岸的情感纠结如心脏之血循环。

**与《碎琉璃》的区别**：
- 《碎琉璃》写的是记忆——童年、故乡、亲情
- 《左心房漩涡》写的是乡愁——失去故乡后四十年的精神漂泊
- 前者是"回忆"，后者是"怀念"
- 前者写"有"，后者写"没有"

**风格分析**：运用现代诗技法（意象切断、节奏跳跃），融合中国古典"赋"的铺陈手法，是王鼎钧散文技法最成熟时期的代表作。

**获奖**：金鼎奖、吴鲁芹散文奖、《中国时报》文学奖推荐奖。

**金句**：
- "故乡只在传说里，只在心上纸上"
- "光天化日，只要我走近它，睁开眼，轰的一声，我的故乡就粉碎了"
- "乡愁是美学，不是经济学"
- "我的乡愁是浪漫而略近颓废的，带着像感冒一样的温柔"

### 4.3 《海水天涯中国人》（1982）

**出版信息**：尔雅出版社，1982年初版。

**核心内容**：游记散文集。以海外华人的视角观察世界，写不同国家中国人的生存状态。书名"海水天涯中国人"本身就是一种漂泊的隐喻——哪里有海水，哪里就有中国人。

### 4.4 《山里山外》（1982）

**出版信息**：洪范书店，1982年初版。与《碎琉璃》是姊妹作。

**核心内容**：续《碎琉璃》的题材，写离开故乡后的流亡经历。书名暗示"山里面"（故乡）和"山外面"（世界）的两个世界。

### 4.5 《情人眼》（1970）

**出版信息**：大林出版社，1970年初版，后改名《情话》。

**核心内容**：抒情散文集，"情人眼"指带着爱意看世界的眼光。王鼎钧所有作品中最温柔、最感性的一本。

### 4.6 《看不透的城市》（1984）

**出版信息**：尔雅出版社，1984年初版。

**核心内容**：都市散文，写纽约等现代都市的观察。与思念乡村/故乡的作品形成对照——"看不透"三个字表达了对都市的矛盾心态。

## 五、回忆录四部曲（1992-2009）

### 5.1 《昨天的云》（1992）

**时间跨度**：1917-1941（出生到抗战初期）

**核心内容**：
第一章：家族史——山东兰陵王氏家族的兴衰
第二章：童年——私塾教育、乡间生活
第三章：抗战爆发——战争的到来打碎了一切
第四章：流亡——与家人分离

**写作特色**：
- 以少年视角写大历史
- 语言温润，但对战争的残酷毫不回避
- "昨天的云" = 往事如云烟，飘走了但曾经存在

### 5.2 《怒目少年》（1995）

**时间跨度**：1942-1945（抗战后期流亡学生时期）

**核心内容**：
- 入国立二十二中，随校辗转安徽、河南、陕西
- 流亡学生的艰苦生活：饥饿、疾病、死亡
- 目睹国民党统治的腐败
- "怒目"的双重含义：对日寇的愤怒 + 少年不甘屈服的眼神

### 5.3 《关山夺路》（2005）

**时间跨度**：1945-1949（国共内战时期）

**核心内容**：
- 从军经历：随宪兵团在沈阳、秦皇岛、天津、青岛、上海等地辗转
- 内战的全景画卷：前线、后方、城市、乡村
- 国共两军的不同面貌
- 跋涉六千七百里的"关山夺路"

**本书特色**：四部曲中最具戏剧张力的一部，充满悬念和紧张感。书名"关山夺路"取自杜甫诗"关山正夺路"。

### 5.4 《文学江湖》（2009）

**时间跨度**：1949-1978（台湾三十年）

**核心内容**：
- 初到台北的困窘
- 进入媒体圈、文学圈的经过
- 与张道藩、林海音、余光中、柏杨、白先勇等人的交往
- 在政治高压下以文学求生存的智慧
- "江湖"隐喻文坛的复杂生态

**完成时间**：2009年3月10日，84岁完稿，历时20年。

**四部曲的整体意义**：以个人经历折射20世纪中国历史，展示"一代中国人的因果纠结、生死流转"。王鼎钧自述写作动机："那些死去的人不会说话了，如果我不替他们说话，他们就白死了。"

## 九、核心思想精华提炼

### 9.1 对文学的根本理解
1. "文章是说话的延长"——文学来源于日常表达
2. "文学的本质是意象"——意象是文学的最小单位
3. "良工式古不违时"——尊重传统但不拒绝创新

### 9.2 对散文的革新
1. 散文可以吸收小说技法和戏剧冲突——"散文小说化"
2. 散文不止抒情，要"感性与知性融合"
3. 散文要有"情节"，不能只是"感想"

### 9.3 对写作教育的贡献
1. "写作可教"——坚决反对天赋论
2. 把写作拆解为可练习的具体技法
3. 以"铺路者"的心态写教学书，不是炫技而是帮助

### 9.4 核心人生观
1. "不要怕，不要悔"——把握当下，不念过往
2. "早临的逆境是福"——挫折是最好的老师
3. "人在苦难中才更像一个人"——苦难是人生的试金石
4. "宽恕不是忘记，是放下了"——成熟的人生态度

## 十、"人生三书"教学应用指南

### 什么年龄段适合读
- 初中生：《开放的人生》——基本的做人道理，容易理解
- 高中生：《人生试金石》——面对考试、升学的挫折
- 大学生/年轻人：《我们现代人》——思考自己与时代的关系

### 怎么用"人生三书"教孩子
1. **每天读一篇**：每篇只有几百字，适合作为睡前阅读
2. **读后讨论**：不考试、不写读后感、只是聊天
3. **联系实际**：问孩子"你今天有没有遇到类似的事？"
4. **不要讲解**：书里的道理已经很简单了，不需要再解释

### 六字箴言的教学示范
"不要怕，不要悔"——这六个字怎么教？
1. 先讲故事——老人送年轻人两句话的故事
2. 再问感受——"你猜那个年轻人后来怎么样了？"
3. 联系生活——"你最近有没有什么让你害怕的事？后悔的事？"
4. 不要上价值——不要升华到"人生真谛"——让孩子自己去想

## 十一、"作文四书"教学应用指南

### 四本书的使用顺序
1. **起点**：《文学种籽》——先理解"文学是什么"，建立基本概念
2. **技法**：《作文七巧》——学会了概念，需要技法落地
3. **解惑**：《作文十九问》——练习中遇到问题，回来查答案
4. **升华**：《灵感》——技法熟练之后，思考"灵感的本质"

### 老师怎么用
- 备课：从《作文七巧》中提取单项练习
- 改作业：用《作文十九问》中的方法来回应学生作品
- 进阶课：根据《文学种籽》的框架设计课程
- 研究：参考《灵感》中的理论

### 学生怎么自修
1. 先读《文学种籽》第一遍——不求全懂，知道大概说什么
2. 再读《作文七巧》——每读一章，做一章的练习
3. 写作时遇到问题——翻《作文十九问》
4. 写了一段时间后——重读《文学种籽》，会有不同的理解
